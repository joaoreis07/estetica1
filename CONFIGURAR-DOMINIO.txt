DOMÍNIO: www.nutrilucianadomingues.com.br
============================================

O código do site já está preparado. Falta configurar em 2 lugares:

────────────────────────────────────────────
1) NO GITHUB (conta joaoreis07)
────────────────────────────────────────────
Abra: https://github.com/joaoreis07/nutriluciana01/settings/pages

a) Branch: gh-pages  |  Folder: / (root)  →  Save
b) Em "Custom domain", digite:
   www.nutrilucianadomingues.com.br
c) Clique Save
d) Marque "Enforce HTTPS" quando aparecer (pode levar algumas horas)

────────────────────────────────────────────
2) NO REGISTRO.DO DOMÍNIO (Registro.br, Hostinger, etc.)
────────────────────────────────────────────
Crie um registro DNS tipo CNAME:

   Nome / Host:     www
   Tipo:            CNAME
   Valor / Destino: joaoreis07.github.io

(O ponto final no destino NÃO é obrigatório na maioria dos painéis.)

Opcional — sem "www" (nutrilucianadomingues.com.br) redirecionar para www:
   No Registro.br use "Redirecionamento" para www, OU registros A do GitHub:
   185.199.108.153
   185.199.109.153
   185.199.110.153
   185.199.111.153

────────────────────────────────────────────
AGUARDE
────────────────────────────────────────────
DNS pode levar de 15 minutos até 48 horas.

Site final: https://www.nutrilucianadomingues.com.br/
